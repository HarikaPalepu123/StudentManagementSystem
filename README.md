# Student Management System (Java)

## Description
This is a Java console application that manages student data using Collections.
It demonstrates sorting using Comparator based on:
- Student ID
- Student Name
- Student Marks

## Technologies Used
- Java
- Java Collections Framework

## Features
- Store student details (ID, Name, Marks)
- Sort students using Comparator
- Console-based output

## How to Run
1. Compile all Java files:
   javac *.java
2. Run the program:
   java StudentManagement

## Purpose
This project is created for learning Java Collections and interview preparation.
